# Writing Prompts

## Content Drafting
Create first drafts for various content types.

```
Write a [CONTENT TYPE] about: [TOPIC]

Target audience: [WHO]
Tone: [FORMAL/CASUAL/TECHNICAL/ETC]
Length: [WORD COUNT OR FORMAT]
Key points to include: [LIST POINTS]

Additional requirements: [ANY SPECIFIC NEEDS]
```

## Editing and Refinement
Improve existing text for clarity and impact.

```
Edit the following text for:
- Clarity and conciseness
- Grammar and punctuation
- Flow and readability
- Tone consistency

Target tone: [SPECIFY]
Keep the core message but improve the delivery.

Text:
[INSERT TEXT]
```

## Email Composition
Craft effective professional emails.

```
Write a professional email for the following situation:

Purpose: [WHAT YOU WANT TO ACHIEVE]
Recipient: [WHO AND THEIR ROLE]
Context: [BACKGROUND INFORMATION]
Key points: [WHAT TO COMMUNICATE]
Desired action: [WHAT YOU WANT THEM TO DO]

Tone: [FORMAL/FRIENDLY/URGENT/ETC]
```

## Summary Writing
Condense long content into concise summaries.

```
Summarize the following content:

Format: [BULLET POINTS/PARAGRAPH/EXECUTIVE SUMMARY]
Length: [TARGET LENGTH]
Audience: [WHO WILL READ THIS]
Focus on: [KEY ASPECTS TO EMPHASIZE]

Content:
[INSERT CONTENT]
```

## Rewriting for Audience
Adapt content for different audiences.

```
Rewrite the following content for a different audience:

Original audience: [WHO IT WAS WRITTEN FOR]
New audience: [WHO SHOULD UNDERSTAND IT NOW]

Adjust:
- Technical level
- Tone and formality
- Examples and analogies
- Length as needed

Content:
[INSERT CONTENT]
```
