# Coding Prompts

## Code Review
Review code for bugs, performance issues, and best practices.

```
Review the following code for:
1. Bugs and potential errors
2. Performance issues
3. Security vulnerabilities
4. Code style and best practices
5. Suggestions for improvement

Language: [LANGUAGE]
Code:
[INSERT CODE]
```


## Refactoring
Improve code structure without changing functionality.

```
Refactor the following code to improve:
- Readability
- Maintainability
- Performance (if applicable)

Keep the same functionality. Explain what changes you made and why.

Language: [LANGUAGE]
Code:
[INSERT CODE]
```
