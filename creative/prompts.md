# Creative Prompts

## Brainstorming
Generate diverse ideas for a given topic or problem.

```
Generate 10 creative ideas for: [TOPIC/PROBLEM]

Context: [RELEVANT BACKGROUND]
Constraints: [ANY LIMITATIONS]

For each idea, provide:
- Brief description
- Key benefit
- Potential challenge
```

## Concept Exploration
Explore a concept from multiple angles and perspectives.

```
Explore the concept of [CONCEPT] from multiple perspectives:

1. Historical context
2. Current applications
3. Future possibilities
4. Contrarian viewpoint
5. Cross-domain connections

Provide unexpected insights or novel combinations.
```

## Story Framework
Create a narrative structure for storytelling.

```
Create a story framework for: [THEME/TOPIC]

Include:
1. Hook/Opening
2. Main characters and their motivations
3. Central conflict
4. Key plot points (3-5)
5. Resolution approach
6. Memorable ending

Genre/Tone: [SPECIFY]
```
