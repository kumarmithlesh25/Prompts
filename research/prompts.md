# Research Prompts

## Topic Deep Dive
Comprehensive exploration of a topic.

```
Provide a comprehensive overview of: [TOPIC]

Cover:
1. Definition and core concepts
2. Historical development
3. Current state and key players
4. Major debates or controversies
5. Future trends
6. Recommended resources for further learning
```

## Literature Review
Synthesize information from multiple sources.

```
Synthesize the following sources on [TOPIC]:

Sources:
[SOURCE 1 SUMMARY]
[SOURCE 2 SUMMARY]
[SOURCE 3 SUMMARY]

Provide:
1. Common themes across sources
2. Contradictions or disagreements
3. Gaps in the research
4. Overall synthesis and conclusions
```

## Fact Verification
Evaluate claims and check for accuracy.

```
Evaluate the following claim:

Claim: [STATEMENT]
Source: [WHERE IT CAME FROM]

Provide:
1. Assessment of claim accuracy
2. Supporting evidence
3. Contradicting evidence
4. Context that affects interpretation
5. Confidence level in your assessment
```

## Concept Comparison
Compare related concepts or theories.

```
Compare and contrast: [CONCEPT A] vs [CONCEPT B]

Cover:
1. Core definitions
2. Key similarities
3. Important differences
4. When to use each
5. Common misconceptions
```
