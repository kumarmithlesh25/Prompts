# Prompts

A collection of prompts organized by use case for various AI tasks.

## Structure

| Folder | Purpose |
|--------|---------|
| `analysis/` | Data analysis, summarization, evaluation |
| `coding/` | Programming, debugging, code review |
| `creative/` | Brainstorming, storytelling, ideation |
| `productivity/` | Planning, task management, workflows |
| `research/` | Investigation, learning, fact-finding |
| `writing/` | Content creation, editing, communication |

## Usage

Each folder contains prompt files for specific tasks. Prompts are designed to be portable across different AI tools.
