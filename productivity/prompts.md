# Productivity Prompts

## Task Breakdown
Break complex tasks into actionable steps.

```
Break down the following task into actionable steps:

Task: [DESCRIBE TASK]
Deadline: [IF APPLICABLE]
Resources available: [LIST RESOURCES]

Provide:
1. Ordered list of steps
2. Dependencies between steps
3. Estimated effort for each step
4. Potential blockers and mitigations
```

## Decision Framework
Structure a decision-making process.

```
Help me make a decision about: [DECISION]

Options I'm considering:
[OPTION 1]
[OPTION 2]
[OPTION 3]

Key factors to consider: [LIST FACTORS]

Provide:
1. Pros and cons for each option
2. Risk assessment
3. Recommendation with reasoning
```

## Meeting Agenda
Create a structured meeting agenda.

```
Create an agenda for a meeting about: [TOPIC]

Meeting details:
- Duration: [TIME]
- Participants: [WHO]
- Goal: [DESIRED OUTCOME]

Include:
1. Time allocations for each item
2. Discussion points
3. Decision items
4. Action items template
```

## Weekly Review
Template for reviewing and planning work.

```
Help me conduct a weekly review.

Last week's goals: [LIST GOALS]
What I accomplished: [LIST ACCOMPLISHMENTS]
Challenges faced: [LIST CHALLENGES]

Provide:
1. Analysis of goal completion
2. Lessons learned
3. Suggestions for next week's priorities
4. Process improvements to consider
```
